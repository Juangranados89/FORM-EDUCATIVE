# Design System — Bienestar Escolar

Principios:
- Amigable para alumnos.
- Claro para orientadores.
- Visual, no clínico.
- Confidencialidad visible.

Componentes:
- AppShell
- Sidebar
- Stepper
- QuestionCard
- AnswerOption
- EmotionButton
- StudentDataForm
- SupportCard
- KPIStatCard
- RiskDistributionChart
- TrendChart
- CourseResultsTable
- WordCloudCard
- AlertBanner

Accesibilidad:
- Contraste AA
- Botones mínimo 44px
- Texto mínimo 16px
- Navegación por teclado
- No depender solo del color
