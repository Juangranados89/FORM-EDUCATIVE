# Pinturas — Vista Mobile

Carpeta destino: `public/assets/mobile/`
Formato: **PNG con fondo transparente**, cuadrado (recomendado 1024×1024, mínimo 512×512).

Para cambiar una ilustración, guarda tu pintura con el **nombre exacto** de la tabla.
Si un archivo no existe todavía, la app simplemente no muestra esa imagen (no se rompe),
así puedes ir suministrándolas una por una.
